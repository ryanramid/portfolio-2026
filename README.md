## 🧠 Sobre o projeto

Este projeto é uma landing page de portfólio pessoal com visual **cyberpunk**, background estilo **Matrix**, terminal fake interativo e detalhes em **light blue**.

A proposta é apresentar um perfil de **desenvolvedor criativo**, unindo tecnologia, análise de dados e design em uma interface moderna, responsiva e visualmente impactante.

🚀 Como executar localmente

Para rodar corretamente no computador, use o Live Server no VS Code.

Passo a passo
# Clone o repositório
git clone https://github.com/ryanramid/portfolio-2026.git

# Acesse a pasta
cd portfolio-2026

Depois:

Abra a pasta no VS Code
Instale a extensão Live Server
Clique com o botão direito em index.html
Selecione Open with Live Server

Como o projeto usa data.json, é recomendado abrir com Live Server para evitar bloqueios do navegador.

🌐 Como publicar no GitHub Pages
Envie os arquivos para um repositório público

Clique em Pages

Selecione a branch main
Escolha a pasta /root
Clique em Save
